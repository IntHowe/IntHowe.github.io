二分查找（一）

2021-04-06 4 min read # 二分查找 # 算法

二分查找要求数据结构必须先排好序，具有有随机访问的特点（例如数组），能够根据中间元素的特点推测它两侧元素的性质，可以在数据规模的对数时间复杂度内完成查找。

二分查找一般写法

public int lower_bound(int[] nums,int first,int last,int value) {
    int right = nums.length,left = 0,mid;
    while(left < right) {
        mid = left + (right - left) / 2;
        if(nums[mid] > mid) {
            right = mid;
        }else {
            left = mid + 1;
        }
    }
    return left;
}
